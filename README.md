# Todo app

An app for creating and managing tasks.

## Requirements

- NodeJS (18.x.x)
- npm (9.x.x)
- MySQL (8.0.35)

## How to run

### Server

To run the server also create and fill a `.env` file (refer to `.env.example`)

```bash
cd server
npm install
npm run dev
```

### Client

```bash
cd client
npm install
npm run dev
```
